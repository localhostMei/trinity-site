Your solution to cloud storage.

Trinity is a free and open source cross platform self hosting file storage service, aimed to make the process as easy to set up and use as possible. No ads, no nonsense.

How cross platform is it?

Trinity is available on all major operating systems. Windows, Linux, MacOS, iOS, Android and more. Head to the download section to see all versions listed.

Features

Easy to set up, with a familiar interface that you don't have to get used to. Being user friendly is our top priority.

Thanks to the service being self hosted, you don't have to ruin the environment by supporting big data centers. Stay eco!

Our amazing community is eager to solve any issues you may have. Go to contacts for our Discord server.

The client app is fully customizable, with easy plugin and theme installing built in.
